# MindOf Liquid Medication Prescribing Calculator

A web-based calculator for MindOf clinicians to calculate liquid medication titration schedules.

## Features

- **Secure Login**: 4-digit PIN authentication (5346)
- **Bottle Size Selection**: Choose from 100ml, 150ml, 200ml, or custom size
- **Titration Phases**: Add multiple titration phases with days and dose
- **Automatic Maintenance Calculation**: Maintenance phase days calculated automatically based on remaining medication
- **Total Days Display**: Shows cumulative total days across all phases
- **Fully Responsive**: Optimized for phones, tablets, and desktops

## Usage

1. Open `index.html` in a web browser
2. Enter the 4-digit PIN: **5346**
3. Select bottle size
4. Add titration phases with number of days and dose (ml per day)
5. Enter maintenance dose - days will be calculated automatically
6. View total number of days

## Technology
